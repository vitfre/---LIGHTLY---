Внешний список

Внешний список по сути представляет из себя окно-пульт. Он позволяет управлять воспроизведением файлов без использования основного окна проигрывателя.

Внешний список создан, чтобы было возможно проводить презентации: на проекторе на полный экран открыто видеоокно, а на мониторе располагается пульт управления этим окном. В результате зрители видят только презентацию и не видят указатель мыши и отвлекающие элементы управления.

Чтобы открыть внешний список, нажмите Alt+P или выберите в общем контекстном меню Окно \ Внешний список.

Можно и вовсе отключить внутренний список и оставить только внешний. Для этого зайдите в Настройки \ Список \ вкладка Вид и переключите на внешний список. В результате всегда будет открываться только внешний список.
