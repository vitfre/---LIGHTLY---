4.8.7

Проигрыватель

  • Настройки->Клавиатура: новая команда в разделе субтитров - *Скопировать текущую строку в буфер обмена* (удобно если вы учите языки, и хотите по быстрому скопировать текущую видимую строку субтитра).

  • MP4: улучшена перемотка по ключевым кадрам.

4.8.0

Проигрыватель

  • Новая шкурка: WMP R.I.P. (автор: Ham).
  • Обновления в движке шкурок: возможность выравнивания элементов по центру, минимальный размер, поддержка битовой прозрачности для ползунков громкости и шкалы времени, несколько новых кнопок, более полный контроль над шкалой времени и бордюром окна, подробнее в разделе "Для творцов" в Истории изменений движка.
  • Всплывающее меню->Звук: добавлена возможность выбора звукового устройства.
  • Всплывающее меню->Список->Разное: опция *пропустить файл*.
  • Список: поддержка сетевых плейлистов которые отдают данные кусками (требуется наличие LAV Filters).
  • Список: возможность выделить все элементы в списке горячей клавишей (по-умолчанию: Ctrl+A).
  • Миниатюры: опция *не показывать окно с настройками миниатюр*.
  • Настройки->Список->Разное: опция *не начинать воспроизведение файла при запуске его из проводника*.
  • Перерисована главная иконка плеера и иконка трея, добавлены большие разрешения.
  • Удалена поддержка Vimeo.

Видео движок

  • Добавлена поддержка воспроизведения MIDI. (чтобы был звук, в Windows\System32 должен находиться банк звуков с именем 28MBGM.SF2, при этом сам банк звуков может быть любым, главное чтобы он назывался так).
  • Встроенная поддержка WavPack.
  • Встроенная поддержка OPUS.
  • Видео декодер: улучшен механизм определения неверного порядка B-Frames.
  • Видео декодер: поддержка воспроизведения H.264-видео с нарушенной последовательностью B-Frames.
  • Видео декодер: обновлен список поддерживаемых VP5/VP6 (nVidia) видеокарт, обновлена информация по видеоадаптерам Intel Haswell.
  • MKV: поддержка тегов CueDuration и CueRelativePosition при перемотке для считывания субтитров.
  • MKV: улучшена совместимость с Lentoid HEVC Decoder.
  • MPEG: поддержка потоковых (закачивающихся в данный момент) файлов, для таких файлов будет обновляться длительность по мере скачивания новых данных, без необходимости перезагружать плеер.
  • MPEG: возможность переключения программ в MPEG-TS.
  • MPEG: улучшена поддержка некоторых AAC-дорожек.
  • AVI: добавлена поддержка H.264 потоков с FourCC "AVC1" и "h264".
  • FLAC: поддержка ID3 v2 тегов (только для изображений).
  • MP4: поддержка A/V-задержек заданных внутри контейнера.
  • MP4: поддержка HEVC-видео созданных через ffmpeg encoder.
  • MP3: поддержка многослойных .MP3 (в которых храняться данные с разными параметрами, актуально для онлайн-трансляций например).
  • MP3: улучшен поиск заголовка (позволяет воспроизводить файлы содержащий мусор/неправильные данные в начале файла).
  • Добавлена поддержка 12 канального звука с неуказанным расположением каналов.
  • Субтитры: улучшения в обработке субтитров, множественные оптимизации и ускорения загрузки/отрисовки субтитров.
  • Субтитры: поддержка анимированных VobSub-субтитров.
  • MP4: оптимизация загрузки файлов.
  • Аудио декодер: для пользователей Windows XP делаем недоступными опции Bitstream.
  • WASAPI: улучшения в проверке поддержки того или иного формата.
  • Субтитры: оптимизация чтения/позиционирования/обработки текстовых файлов.
  • Обновлены кодеки.
