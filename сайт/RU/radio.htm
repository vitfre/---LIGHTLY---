Интернет радио

Light Alloy умеет проигрывать онлайн радио. Кроме того проигрыватель имеет специальный диалог, позволяющий выбрать для себя одну из тысяч радиостанций со всего мира. Называется этот диалог "Каталог интернет радиостанций"

Каталог интернет радиостанций

Открыть каталог интернет радиостанций можно несколькими способами:

  • Через контекстное меню кнопки Добавить в список;
  • Через Общее контекстное меню \ Файл;
  • Клавиатурным сочетанием Ctrl+F2;

Это окно включает в себя панель управления и два режима:

  • Избранное;
  • IceCast.

Панель управления позволяет управлять проигрыванием выбранного радио. Она дублирует кнопки основного окна, которое в это время заблокировано.

В режиме IceCast загружается из интернета список радиостанций (примерно 2 Мб). Он содержит тысячи позиций со всего мира. Среди этого множества станций можно подобрать интересующие и перенести их в Избранное (через контекстное меню).

Режим Избранное представляет из себя список радиостанций, составленный пользователем лично. Радиостанции в избранное можно добавить несколькими способами:

  • копированием из режима IceCast;
  • копированием выделенных записей из плейлиста (через контекстное меню кнопки +);
  • с помощью загрузки или добавления списка ссылок (через контекстное меню кнопки +; доступны форматы .aimppl, .rsd, собственный формат .lau);
  • Вводом вручную (с помощью кнопки на панели управления + или горячей клавиши Insert);

Чтобы прослушать радиостанцию в любом из двух режимов, достаточно произвести по ней двойной щелчок, нажать Enter или нажать кнопку Проиграть выбранный на панели управления.

Дополнительные возможности каталога интернет радиостанций

Проверка ссылок на актуальность

Работает только в режиме "Избранное".

  1. Выделяем нужные станции.
  2. В контекстном меню кнопки - выбираем пункт "проверить существование выбранных станций".

Недоступные станции станут серыми.

Также может быть полезна галка "Обновлять название при воспроизведении" в Настройках каталога интернет радиостанций. Если она активна, то при воспроизведении название станции будет по возможности браться с сервера станции. Если не заполнены данные в колонках "Формат" и "Битрейт", они так же будут заполняться с сервера.

Навигация

Для перехода по перечню станций можно использовать:

  • + / – - для перехода на следующую/предыдущую станцию;
  • Ctrl++ / Ctrl+– - для прыжка с шагом в 10 станций;
  • Alt++ / Alt+– - для прыжка с шагом в 100 станций.

Для быстрого перехода к конкретной строке можно использовать цифровой набор:

  • 1-9 - обычным нажатием на цифровой клавиатуре справа;
  • 10-99 - удерживая Ctrl;
  • 100-999 - удерживая Alt.

Поиск

Поиск работает по колонкам "Название" "Жанр".

Если поле поиска подсвечено желтым, это значит, что выведен список только тех станций, которые соответствуют поиску.

Фильтрация

Существуют такие методы фильтрации:

  • с помощью настроек фильтрации;
  • переключением показа/скрытия разделов радиостанций.

Настройки фильтрации

Настройки фильтрации вызываются через контекстное меню Настроек каталога интернет радиостанций.

Они позволяют:

  • скрывать строки с заявленными словами в названиях и адресах;
  • показывать только определенный формат потока (mp3, aac и т.д.);
  • отображать только те станции, которые соответствуют критериям к минимуму и максимуму битрейта (например, от 96 до 256 кбит/с).

Разделы радиостанций

Каждую радиостанцию можно отнести к одному из семи разделов. Сделать это можно через форму редактирования или через контекстное меню кнопки Искать. Разделы можно произвольно называть и присваивать им уникальный цвет. Также, через контекстное меню кнопки Искать можно указать, какие именно разделы будут отображаться в окне. Быстрее это можно сделать через Настройки каталога интернет радиостанций, сняв или установив там галки напротив нужных разделов.

Кроме того, каждый раздел может иметь свой цвет, заданный в Настройках каталога интернет радиостанций.

Запись

Light Alloy позволяет записывать радиостанции в файл. Возможна запись как непрерывным файлом, так и с разбивкой по трекам. Определить это, а также указать шаблон для имени файлов и место его сохранения можно в Настройках \ Звук \ вкладка Радио.
